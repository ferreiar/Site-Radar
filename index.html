ARDUINO UNO

Arduino Uno é uma placa de microcontrolador. Ela possui 14 pinos digitais, 6 entradas analógicas, e se comunica via USB com um computador para programação. O Arduino Uno é compatível com uma grande de sensores

LED

O LED no Arduino é um componente que emite luz e serve para indicar estados ou feedback visual. É usado em projetos para ensinar sobre controle de saída digitais, permitindo ligá-lo e desligá-lo facilmente.

PROTOBOARD

O protoboard é uma placa de ensaio que permite fazer conexões temporárias entre componentes eletrônicos sem solda, facilitando a criação e testes de circuitos.

Sensor Ultrassônico

utilizado em projetos com Arduino para medir distâncias. Ele emite ondas sonoras de alta frequência e calcula a distância com base no tempo que leva para o eco retornar.

Servo Motor

Um servo motor é um dispositivo que permite o controle preciso da posição angular. Gira em um arco limitado, geralmente de 0 a 180 graus> Não permite controle de permissão
